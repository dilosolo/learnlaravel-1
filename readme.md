
## Deskripsi Kegiatan
Inisiasi DILo Solo dan PHP Indonesia dalam rangka untuk membantu teman-teman yang mau belajar Pemrograman Web.

## Cara Install
- Download via .zip / ssh
- Buka folder learnlaravel-1 di terminal
- Ketik "composer install"
